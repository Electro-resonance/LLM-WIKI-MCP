# Books and Long Chapters

Example Books and Long Chapters used for testing the LLM WIKI and used with the various docs to form a starting point from which the LLM WIKI can build it’s own self-guide.

| Title                                                          | File                                             | Purpose                                                                                                                |
|----------------------------------------------------------------|--------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| The Self-Extending LLM Wiki                                    | \<THE_SELF_EXTENDING_LLM_WIKI.md\>               | A practical/philosophical book-length guide to the project architecture and design direction.                          |
| The Living LLM Wiki                                            | \<THE_LIVING_LLM_WIKI.md\>                       | A short introductory book that explains the wiki, MCP layer, vault, and practical philosophy.                          |
| The Living LLM Wiki Development Chronicle                      | \<THE_LIVING_LLM_WIKI_DEVELOPMENT_CHRONICLE.md\> | A long-form development chronicle.                                                                                     |
| The Tool That Asks Itself                                      | \<THE_TOOL_THAT_ASKS_ITSELF.md\>                 | A long chapter explaining how an MCP/wiki tool can inspect its own configuration, usage, limits, and evidence surface. |
| The Self-Extending LLM Wiki: Toward Reflective Agentic Systems | \<../BOOK_SELF_EXTENDING_AGENTIC_SYSTEMS.md\>    | Reflective book.                                                                                                       |
